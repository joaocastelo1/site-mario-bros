# 🎮 Super Mario Bros - Fã Site

Este é um projeto simples de site desenvolvido com **HTML**, **CSS** e **JavaScript** como homenagem ao filme **Super Mario Bros (2023)**. O objetivo é praticar estruturação de páginas web, estilização e interação com o usuário usando JavaScript puro.

## ✨ Funcionalidades

- 🎞 Exibição do trailer do filme via YouTube
- 🧾 Sinopse do filme
- 🔘 Botão para exibir informações adicionais (diretores, gênero, lançamento)
- 📱 Layout responsivo e visual inspirado no universo Mario
